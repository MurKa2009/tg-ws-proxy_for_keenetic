# TG WS Proxy для Keenetic (Entware)

Скрипт для автоматической установки и настройки [TG WS Proxy](https://github.com/Flowseal/tg-ws-proxy) на роутерах Keenetic с Entware.
Этот прокси ускоряет работу Telegram и помогает обходить блокировки, упаковывая MTProto-трафик в зашифрованные WebSocket (TLS) соединения.

---

## ✨ Особенности версии для Keenetic
- **Автоматизация**: Скрипт самостоятельно устанавливает Python, зависимости и настраивает сервис.
- **Оптимизация**: Настроен на минимальное потребление RAM (важно для роутеров).
- **Автозапуск**: Прокси запускается автоматически при старте роутера через `/opt/etc/init.d/`.
- **Гибкость**: Выбор IP-адреса (локальный/внешний) во время установки.

---

## 📋 Предварительные требования
- Установленная среда **Entware** на Keenetic.
- Доступ к интернету через роутер.
- Утилита `curl` (если отсутствует, см. инструкцию ниже).

---

## 📦 Установка `curl` (если отсутствует)

```bash
opkg update
opkg install curl ca-certificates
```

---

## 🚀 Установка TG WS Proxy

### Шаг 1. Запустите установку
```bash
curl -sL https://raw.githubusercontent.com/MurKa2009/tg-ws-proxy_for_keenetic/main/install.sh | sh
```

### Шаг 2. Следуйте подсказкам скрипта
- Выберите IP-адрес (локальный или внешний).
- Дождитесь завершения установки.
- **Сохраните выданный Secret!**

### Шаг 3. Настройте Telegram
- **Тип**: MTProto
- **Сервер**: IP вашего роутера (обычно `192.168.1.1`)
- **Порт**: `1443`
- **Секрет**: `dd` + ваш сгенерированный секрет

---

## ⚙️ Управление сервисом

| Команда | Описание |
|---------|----------|
| `/opt/etc/init.d/S99tgwsproxy start` | Запустить прокси |
| `/opt/etc/init.d/S99tgwsproxy stop` | Остановить прокси |
| `/opt/etc/init.d/S99tgwsproxy restart` | Перезапустить прокси |
| `tail -f /opt/var/log/tgwsproxy.log` | Посмотреть логи |
| `ps aux \| grep tg-ws-proxy` | Проверить, работает ли прокси |

---

## 🔥 Проброс порта (для доступа из интернета)

1. Зайдите в веб-интерфейс Keenetic (`192.168.1.1`).
2. Перейдите в раздел **"Межсетевой экран" (Firewall)**.
3. Добавьте правило проброса порта:
   - **Внешний порт**: `1443`
   - **Внутренний порт**: `1443`
   - **Внутренний IP**: IP вашего роутера (обычно `192.168.1.1`)
   - **Протокол**: `TCP`
4. Сохраните настройки.

Теперь подключаться к прокси можно по вашему внешнему IP-адресу.

## 🆘 Удаление прокси
   ```bash
   /opt/etc/init.d/S99tgwsproxy stop
   rm -rf /opt/tg-ws-proxy
   rm /opt/etc/init.d/S99tgwsproxy
   ```
